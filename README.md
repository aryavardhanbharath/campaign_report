# Digital Ad Campaign Dashboard

A simple dashboard built using **HTML, CSS, and JavaScript** to analyze digital ad campaign performance.

## Features
- Calculates CTR, CPC, CPM, and Conversion Rate
- Provides performance rating using conditional logic
- Visualizes campaign metrics using Chart.js
- Clean and simple UI for entering campaign data

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Chart.js

## About
This project is inspired by real campaign optimization tasks handled during my internship at **Mediamint**, where I worked as a Campaign Manager Intern.
